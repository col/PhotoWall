# AR Photo Wall

# Pre-Requisites
- XCode 10
- Ruby 2.4.0

# Development
- `bundle`
- `pod install`
- `fastlane match development`
- `cp PhotoWall/Secrets_template.plist PhotoWall/Secrets.plist`
- Add the AWSSecretKey and AWSAccessKey values to `Secrets.plist`
